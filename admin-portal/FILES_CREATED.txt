To Build & Run:
  1. npm install
  2. npm run dev        (development server on :5173)
  3. npm run build      (production build to dist/)
  4. npm run preview    (preview production build)

Production Deployment:
  - npm run build
  - Copy dist/ contents to server at /admin/ path
  - API calls automatically go to /api/ on same server
  - Ready to serve with any static file server
